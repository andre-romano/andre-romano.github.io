# Projects & Software Tools

Here are some of my technical productions:

* [File Conversor (2025):](https://github.com/andre-romano/file_conversor) A modern file conversion utility for various formats (audio, video, image, pdf, doc, xls, ppt, among others). 
* [Python PDM template (2024):](https://github.com/andre-romano/python_pdm_template) A template for Python projects using the PDM package manager, VSCode, Pytest, Coverage, Pylance, Ruff. 
* [Podman NDNSim Container (2024):](https://github.com/andre-romano/podman-ndnsim) A containerized environment for simulating Named Data Networking (NDN) using Podman. 
* [IFBA Valença LaTeX Template (2023):](https://github.com/andre-romano/ifba_val_latex) A LaTeX template for the IFBA Valença campus. 
* [P4_16 Language Support for VSCode (2019):](https://github.com/andre-romano/vscode_p4_16) A syntax and language support extension for the P4 data plane programming language. 

For more source code and active developments, visit my [GitHub Profile](https://github.com/andre-romano).
